este es el codigo hijos de la verga, lo que no esta en la carpeta es el backend.

lo siguiente son los comandos que deben meter.

--------------------------------------------------------------

Inicializa tu proyecto (si aún no lo has hecho):
Abre tu terminal en la carpeta de tu proyecto y escribe:
//////////////////////
npm init -y
/////////////////////
Esto creará el archivo package.json base.
-------------------------------------------------------------
Instala las dependencias:
Ejecuta el siguiente comando para instalar todas estas librerías de una vez:

/////////////
npm install cors dotenv express mysql2
////////////



Esto descargará los paquetes necesarios en una carpeta llamada node_modules y actualizará automáticamente tu archivo package.json para incluirlos en la sección dependencies
